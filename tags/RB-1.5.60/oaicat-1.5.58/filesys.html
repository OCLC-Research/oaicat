File System Implementation

A file system implementation is included with the OAICat package. The purpose is to provide OAI access to metadata records stored in files in the server's directory structure.

To test the file system implementation, follow these steps:

  1. Copy oaicat.war to Tomcat's webapps directory, restart Tomcat, and test the default configuration.
  2. Restart Tomcat and test.

If this works, change the values in webapps/oaicat/WEB-INF/oaicat.properties to suit your needs.
